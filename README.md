# Lead Generator

Lead Generator is a modern Python desktop application that helps users generate public business lead lists by searching online sources based on a selected category, location, and speed mode.

The app extracts publicly available business contact information such as websites, email addresses, and phone numbers, then exports the results to a CSV file.

## Features

- Modern dark-mode desktop interface
- Search by business category and location
- Selectable search modes:
  - Slow Mode: better accuracy
  - Fast Mode: balanced speed and accuracy
  - Instant Mode: maximum speed
- Extracts public business emails
- Extracts public phone numbers
- Saves website/source links
- Live activity log
- Lead counter
- CSV export
- Built with Python and CustomTkinter
